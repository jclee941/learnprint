# HYCU 학습 이력서 (learnprint)

> 흩어진 학습 경험을 **역량·증거 기반 학습 이력서**로 자동 재구성하는 웹앱 — 입력 → 역량별 그룹화 → 내보내기·인쇄까지 한 흐름으로. (AI 학습 에이전트는 분석·코칭·면접 대비를 돕는 보조 기능)

**🔗 라이브 데모**: <https://learnprint.jclee.me> — 설치 없이 브라우저에서 바로 체험할 수 있습니다 (첫 실행 시 HYCU 수강 과목 시드 자동 주입).

## 프로젝트 개요

사이버대학교에서 쌓은 강의·과제·프로젝트·자격증·독학·대회 등 **흩어진 학습 경험을 한 곳에 입력하면**, 결정론적 규칙 엔진이 이를 **6개 핵심 역량(+ 기타 학습 경험 폴백)으로 자동 분류하고 각 학습 흔적을 "증거(evidence)"로 연결한 학습 이력서**로 재구성합니다. 결과는 Markdown·JSON·증거 원장(Evidence Ledger)으로 내보내거나 브라우저 인쇄로 깔끔하게 출력할 수 있습니다.

규칙 엔진은 네트워크·AI 없이도 항상 동작하며 **동일 입력 → 동일 결과**를 보장합니다. 그 위에 선택적으로 **LLM 기반 AI 학습 에이전트**를 붙여 역량 자기소개 작성·학습 코칭·예상 면접질문 생성을 실시간 스트리밍으로 제공합니다.

2026학년도 HYCU AI 학습법 공모전 부문 5 "AI 또는 바이브코딩으로 나만의 학습도구 제작" 출품작입니다.

## 사용 흐름

1. **학습 경험 입력** — 강의·과제·프로젝트·자격증 등을 카테고리·기간·증거와 함께 기록
2. **이력서 생성** — 규칙 엔진이 6개 역량(+ 기타 학습 경험 폴백)으로 자동 분류·정렬
3. **증거 기반 이력서 확인** — 역량별로 묶인 학습 흔적을 증거 문장·출처와 함께 검토
4. **내보내기·인쇄** — Markdown / JSON / 증거 원장 Markdown 다운로드 또는 브라우저 인쇄
5. **(선택) AI 코칭** — 등록한 이력을 컨텍스트로 자기소개·코칭·면접질문 생성

## 공모전 제출 포인트

- **분산된 학습 경험을 하나의 증거 자산으로** — 사이버대 학습자가 여기저기 흩어 둔 학습 흔적을 취업·면접에 쓸 수 있는 역량·증거 이력서로 전환
- **설명 가능한 결정론적 엔진** — 동일 입력에 항상 동일 결과. 어떤 학습 경험이 어떤 역량으로 연결됐는지 사용자가 직접 검토·수정할 수 있어, 심사·포트폴리오 재현 시 결과가 흔들리지 않음
- **AI는 보조로만** — 핵심 기능(이력서 재구성·내보내기·인쇄)은 AI·네트워크 없이도 완전히 동작하고, AI 에이전트는 그 위에 얹는 선택적 코칭 레이어
- **실제 데이터·실제 배포** — 작성자의 실제 HYCU 수강 과목 시드 내장, 공개 도메인(<https://learnprint.jclee.me>)에서 구동

## 핵심 기능

- **학습 경험 입력**: 강의·과제·프로젝트·자격증·독학·대회·기타 카테고리로 학습 흔적과 증거(링크·메모)를 기록
- **역량·증거 기반 이력서 자동 생성**: 입력한 학습 경험을 6개 역량(데이터·AI 활용 / 문제해결·논리력 / 협업·커뮤니케이션 / 자기주도학습 / 전공·직무역량 / 기획·문서작성)으로 분류하고, 어디에도 안 잡히는 흔적은 기타 학습 경험(폴백)으로 남김. 각 항목을 증거 문장·출처와 함께 역량 그룹으로 묶음. 키워드 매칭 기반 결정론적 엔진이라 동일 입력에 항상 동일 결과
- **내보내기·클립보드 복사**: 이력서를 Markdown·JSON·증거 원장(Evidence Ledger) Markdown으로 다운로드하거나 클립보드로 복사 (역량/증거/항목 직렬화, Markdown 복사·JSON 복사·증거 원장 복사 버튼 제공)
- **인쇄**: `@media print`로 입력·버튼을 숨기고 이력서 문서만 깔끔하게 출력
- **HYCU 실제 수강 과목 시드**: 첫 실행 시 작성자의 실제 HYCU 과목 기반 학습 경험이 자동 주입되어 바로 결과 확인 가능 (개인정보·학번 미포함)
- **데이터 유지**: 브라우저 localStorage에 저장되어 새로고침 후에도 유지

### 선택 기능

- **AI 학습 에이전트**: 등록한 학습 이력을 컨텍스트로 주입해 ① 역량 분석·자기소개 문단 작성 ② 학습 코칭 챗봇 ③ 예상 면접질문 생성 — 실시간 스트리밍 응답. AI 연결이 없으면 규칙 엔진 결과만으로도 완전히 동작

## 기술 스택

- React 19 + Vite 6 + TypeScript (strict)
- 결정론적 역량 분석 엔진 (`src/lib/ai/analyzer.ts`) — 키워드 매칭 기반, 의존성 없음
- Node.js `node:http` BFF — AI 에이전트용 OpenAI 호환 LLM(cliproxy) SSE 스트리밍 중계
- Vitest — 115개 테스트 통과 (단위·컴포넌트·서버)

## 설치 및 실행

```bash
npm install

# UI 개발 서버 (HMR). /api 는 BFF(4173)로 프록시됨
npm run dev

# 빌드 + 정적 서빙(+AI 에이전트 BFF, 포트 4173)
npm run build && npm run serve

# 테스트 / 타입체크
npm test
npm run typecheck
```

규칙 엔진 기반 이력서 생성·내보내기·인쇄는 별도 설정 없이 바로 동작합니다. (선택) AI 학습 에이전트를 쓰려면 `.env.example`을 `.env`로 복사해 LLM 엔드포인트를 설정하세요 — 자세한 환경변수는 `.env.example` 참고.

## 개인정보 및 면책

- 본 프로젝트는 교육 목적의 출품작입니다.
- 학습 데이터는 브라우저 localStorage에만 저장되며 개인정보·학번·인증정보는 수집하지 않습니다.
- AI 학습 에이전트를 사용할 경우에 한해, 입력한 학습 이력 일부가 서버 BFF(`/api/agent/chat`)를 거쳐 LLM API로 전송될 수 있습니다. 민감정보는 입력하지 않는 것을 원칙으로 합니다.
- AI 연결이 없거나 사용하지 않을 때는 결정론적 규칙 엔진만으로 이력서 생성·내보내기·인쇄가 완전히 동작합니다.
