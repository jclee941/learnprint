## 작품개요

HYCU 학습 이력서(learnprint)는 흩어진 전공 학습 경험을 AI 학습 코치로 재구성하여 역량·증거 기반 이력서로 만드는 학습법과, 이를 기록하고 검증하는 웹앱이다. 사용자는 강의나 실습 후 내 언어로 요약하고, 이해가 안 가는 부분을 AI에게 구체적으로 질문하여 개념을 재구성하며, 그 과정에서 나온 질문 로그와 오답 노트, 성찰문 등을 learnprint에 기록한다. learnprint는 규칙 기반 분석 엔진으로 6개 과목의 학습 기록을 데이터·AI 활용, 문제해결·논리력, 협업·커뮤니케이션, 자기주도학습, 전공·직무역량, 기획·문서작성 등 6개 역량 그룹으로 분류한 이력서를 생성한다. 네트워크가 없을 때도 결정론적 규칙 기반 오프라인 폴백이 동작하여 학습 데이터가 사라지지 않으며, AI 연결 시에는 서버 측 BFF를 통해 cliproxy LLM API와 통신하는 AI 에이전트가 역량 분석, 학습 코칭, 예상 면접 질문을 실시간 스트리밍으로 제공한다. 생성된 이력서는 Markdown과 JSON 파일로 내보내거나 브라우저 인쇄 기능으로 출력할 수 있으며, 모든 학습 데이터는 브라우저 localStorage에 저장되어 새로고침 후에도 유지된다. 본 앱은 https://learnprint.jclee.me 에서 실제 구동하며, 6개 실제 수강 과목을 기반으로 한 역량 분류 결과를 산출물로 보여준다.
