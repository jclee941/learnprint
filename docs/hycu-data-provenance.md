# HYCU 학습 데이터 작성 기준

> learnprint의 초기 학습 경험 데이터(`src/data/hycu-seed.json`)가 어떤 기준으로 작성되었는지 투명하게 밝힙니다.

---

## 1. 데이터 출처

- 초기 6건의 학습 경험은 **작성자의 HYCU 실제 수강 과목명**을 기반으로 합니다.
- 과목별 학습 내용(설명, 증거, 역량 분류 등)은 **작성자가 직접 작성**한 것이며, LMS 시스템에서 자동으로 가져온 원본 데이터가 아닙니다.
- 데이터 작성 시 개인정보·학번·인증정보·세션·API key는 일체 포함하지 않았습니다.

## 2. 포함 / 제외 필드

산출물(`hycu-seed.json`)에는 다음만 포함합니다.

| 포함된 필드 | 예시 |
|------|------|
| 과목명 | "컴퓨터구조론" |
| 학습 유형 | "강의" |
| 학기 | "2026-1학기" |
| 학습 내용 요약 | "컴퓨터 구조의 기본 원리와 CPU 동작 방식 학습" |
| LMS 과목 링크 | `https://lms.hycu.ac.kr/classroom/index.do?crsCreCd=...` |

| **의도적으로 제외된 항목** | 사유 |
|------|------|
| 이름·학번 | 개인 식별정보 |
| 세션 쿠키 / 토큰 | 인증 정보 — 절대 미포함 |
| `.env` / API Key | 계정·비밀 정보 — 절대 미포함 |

## 3. 외부 도구와의 관계

- 본 데이터는 **자동 출석 도구, LMS 연동 프로그램, 외부 파싱 도구와 무관**합니다.
- 과목명은 작성자의 실제 수강 이력을 참고했으나, 학습 내용과 증거는 직접 기술한 창작물입니다.
- `src/data/hycu-seed.json` 및 관련 파일에는 쿠키·토큰·`.env` 값·학번·이름이 포함되어 있지 않습니다.
